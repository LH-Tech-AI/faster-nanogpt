# faster-nanogpt
Optimized nanoGPT evolution using the Muon optimizer for 2x faster convergence. Features a modern Llama/Gemma-style architecture: RoPE, RMSNorm, QK-Norm, and ReLU² activation. Designed for maximum sample efficiency and stability on modern Blackwell/Ada GPUs. bfloat16 &amp; torch.compile ready. The fastest way to train small-to-medium GPTs.
